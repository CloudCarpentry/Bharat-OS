# Documentation Archive

This folder stores superseded or low-signal documents that are kept for historical traceability, not as active guidance.

## Archive policy

A document should move here when one or more apply:

- it was explicitly marked as archived/superseded,
- it contains brainstorming content that is no longer part of the active plan,
- it is a backup/temporary artifact (for example `*.orig`),
- it materially conflicts with current repository structure and has been replaced by an active canonical document.

## Restoring a document

If a document becomes relevant again:

1. move it back under an active docs path,
2. update links in `docs/README.md` and relevant local indexes,
3. add a short note in the commit message describing why it is active again.
